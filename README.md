## Installation

### From within the root directory:

```sh
sudo npm install -g bower
npm install
bower install
```


## Usage
  Clone or download the repo and run npm install, then open three terminal windows:
1. npm run server-dev
2. npm run react-dev
3. sudo mongod


## Requirements

Node ^6.11.4
MongoDb
React
